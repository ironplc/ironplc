==============
Source Formats
==============

IronPLC supports multiple source file formats for IEC 61131-3 programs:

- **Text** (``.st``, ``.iec`` files) - Native Structured Text format for developing new code
- **PLCopen XML** (``.xml`` files) - Interchange format for importing programs from other environments
- **TwinCAT** (``.TcPOU``, ``.TcGVL``, ``.TcDUT`` files) - Beckhoff TwinCAT 3 project files
