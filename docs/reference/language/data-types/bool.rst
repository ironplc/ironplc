====
BOOL
====

Boolean value: ``TRUE`` or ``FALSE``.

.. list-table::
   :widths: 30 70

   * - **Size**
     - 1 bit
   * - **Values**
     - ``TRUE``, ``FALSE``
   * - **Default**
     - ``FALSE``
   * - **IEC 61131-3**
     - Section 2.3.1
   * - **Support**
     - Supported

Literals
--------

.. code-block:: iec61131

   TRUE
   FALSE
   BOOL#1
   BOOL#0
