===========
TIME_OF_DAY
===========

Time of day value.

.. list-table::
   :widths: 30 70

   * - **Default**
     - ``TOD#00:00:00``
   * - **IEC 61131-3**
     - Section 2.3.1
   * - **Support**
     - Not yet supported

Literals
--------

.. code-block::

   TOD#14:30:00
   TIME_OF_DAY#08:00:00
   TOD#23:59:59.999
