====
UINT
====

16-bit unsigned integer.

.. list-table::
   :widths: 30 70

   * - **Size**
     - 16 bits
   * - **Range**
     - 0 to 65,535
   * - **Default**
     - 0
   * - **IEC 61131-3**
     - Section 2.3.1
   * - **Support**
     - Supported

   UINT#65535
   UINT#16#FFFF
