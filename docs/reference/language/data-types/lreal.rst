=====
LREAL
=====

64-bit double-precision IEEE 754 floating-point number.

.. list-table::
   :widths: 30 70

   * - **Size**
     - 64 bits
   * - **Range**
     - Approximately -1.8E+308 to 1.8E+308
   * - **Precision**
     - ~15 decimal digits
   * - **Default**
     - 0.0
   * - **IEC 61131-3**
     - Section 2.3.1
   * - **Support**
     - Not yet supported

Literals
--------

.. code-block::

   LREAL#3.14159265358979
   LREAL#-1.0
   LREAL#1.0E+100
