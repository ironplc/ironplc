====
BYTE
====

Bit string of 8 bits.

.. list-table::
   :widths: 30 70

   * - **Size**
     - 8 bits
   * - **Range**
     - 16#00 to 16#FF
   * - **Default**
     - 16#00
   * - **IEC 61131-3**
     - Section 2.3.1
   * - **Support**
     - Not yet supported

Literals
--------

.. code-block::

   BYTE#16#FF
   BYTE#2#11001010
   BYTE#8#377
