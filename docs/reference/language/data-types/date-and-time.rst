=============
DATE_AND_TIME
=============

Combined date and time of day value.

.. list-table::
   :widths: 30 70

   * - **Default**
     - ``DT#0001-01-01-00:00:00``
   * - **IEC 61131-3**
     - Section 2.3.1
   * - **Support**
     - Not yet supported

Literals
--------

.. code-block::

   DT#2024-01-15-14:30:00
   DATE_AND_TIME#2024-12-31-23:59:59
