====
SINT
====

8-bit signed integer.

.. list-table::
   :widths: 30 70

   * - **Size**
     - 8 bits
   * - **Range**
     - -128 to 127
   * - **Default**
     - 0
   * - **IEC 61131-3**
     - Section 2.3.1
   * - **Support**
     - Supported
