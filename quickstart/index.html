Prerequisites#

The first step is to install Visual Studio Code. Go to Visual Studio Code then follow the steps for your platform.

Installation#

Follow the steps below to install IronPLC.

Install IronPLC CLI

  1. Download the latest IronPLC MSI installer ironplcc-x86_64-windows.msi from IronPLC GitHub releases.

  2. Run the MSI installer and follow the prompts to complete installation of the CLI.

Install IronPLC Visual Studio Code Extension

  1. Download the latest IronPLC Visual Studio Code Extension ironplc-vscode-extension.vsix from IronPLC GitHub releases.

Run Visual Studio Code, then in Visual Studio Code:

  1. Go to the Extensions view by clicking on the Extensions icon in Activity Bar on the side of VS Code or using the View: Extensions command (Ctrl+Shift+X).

  3. In the Install from VISX dialog, select the VISX file you downloaded earlier.
